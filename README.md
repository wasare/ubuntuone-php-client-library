This is a library written in PHP to interact with Ubuntu One APIs that you can find here https://one.ubuntu.com/developer/.  


### Requirements: ###

- PHP 5.1 or newer
- json extension for PHP
- cURL extension for PHP
- OAuth PECL package for PHP
